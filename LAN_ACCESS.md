# DjangoBlog 局域网访问指南

**启动时间：** 2026-03-19  
**服务器地址：** http://0.0.0.0:8000/

---

## 🌐 访问地址

### 本机访问
```
http://127.0.0.1:8000/
http://localhost:8000/
```

### 局域网访问
```
http://192.168.1.2:8000/
```

### 其他设备访问（同局域网内）
在手机、平板或其他电脑的浏览器中输入：
```
http://192.168.1.2:8000/
```

## 🔑 管理员账号

**后台登录：** http://192.168.1.2:8000/admin/

如果忘记密码，可以运行：
```bash
python manage.py createsuperuser
```

---

## ⚠️ 注意事项

1. **防火墙**：确保 Windows 防火墙允许 8000 端口
2. **同一网络**：其他设备需要和电脑在同一 WiFi/局域网
3. **保持运行**：关闭命令行窗口会停止服务器
4. **开发模式**：当前是开发环境，不要用于生产

---

## 🛑 停止服务器

在命令行窗口按 `Ctrl + C` 即可停止
