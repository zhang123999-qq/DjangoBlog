# 默认头像文件夹

此文件夹存放用户默认头像。

## 头像列表

当前包含 **20** 个不同风格的动漫头像：

| 文件名 | 风格 |
|--------|------|
| anime-1.png ~ anime-12.png | Adventurer 动漫风格 |
| anime-13.png ~ anime-14.png | Big Smile 卡通风格 |
| anime-15.png ~ anime-16.png | Fun Emoji 表情风格 |
| anime-17.png ~ anime-18.png | Bottts 机器人风格 |
| anime-19.png ~ anime-20.png | Lorelei 像素风格 |

## 使用说明

### 自动分配

新用户注册时，系统会自动从这些头像中随机分配一个。

### 手动更换

用户可以在个人资料页面手动更换头像。

## 添加新头像

1. 将头像文件放入 `static/img/avatars/` 和 `media/avatars/` 目录
2. 支持格式：PNG、JPG、JPEG、GIF
3. 推荐尺寸：200x200 像素
4. 推荐文件大小：< 20KB

## 头像来源

头像通过 [DiceBear](https://dicebear.com/) API 生成，基于不同的种子值创建独特的动漫风格头像。
