## 🟡 性能优化（建议改进）

### 1. 粒子动画性能
**文件：** `static/js/tech-effects.js`
**问题：** 粒子动画在低端设备上可能造成卡顿
**优化建议：**
- 使用 `will-change: transform` CSS 属性
- 减少粒子数量（根据设备性能动态调整）
- 使用 `requestAnimationFrame` 节流

### 2. 图片懒加载
**问题：** 页面图片未实现懒加载
**优化建议：**
```html
<img src="..." loading="lazy" alt="...">
```

### 3. CSS 文件大小
**问题：** CSS 文件较大（约 13KB）
**优化建议：**
- 生产环境启用 CSS 压缩
- 使用 PurgeCSS 移除未使用样式
- 分割关键 CSS 和非关键 CSS
