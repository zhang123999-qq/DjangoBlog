# 异常处理分层收敛（阶段 17：审核中心指标面板）

覆盖文件：
- `moderation/templates/moderation/dashboard.html`

## 目标

将阶段 16 的 metrics API 结果直接可视化到审核中心页面，降低排障成本。

## 实现

1. 在审核中心顶部新增“审核保护指标（最近 10 分钟）”卡片
2. 展示关键指标：
   - 总请求
   - 限流命中
   - 并发命中
   - 并发峰值
   - 通过成功/失败
   - 拒绝成功/失败
3. 展示热点用户（Top3）
4. 页面初始化拉取一次，之后每 30 秒自动刷新一次

## 设计说明

- 指标拉取失败时静默降级，不影响审核主流程。
- 面板仅消费 `/api/moderation/metrics/`，无额外后端写入。
