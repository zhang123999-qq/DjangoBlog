# 上传 UI 反馈（进度条 + 状态文案）

已在以下编辑器脚本接入上传 UI 反馈：

- `static/js/editor-init.js`
- `static/js/admin-editor.js`

## 行为说明

当通过附件选择器上传文件时：

1. 打开 TinyMCE 原生进度态（`setProgressState(true)`）
2. 轮询状态时显示提示：
   - `pending` → 文件已上传，等待扫描...
   - `scanning` → 文件安全扫描中...
   - `ready` → 扫描通过，正在完成上传...
3. 上传成功显示绿色成功提示
4. 失败显示红色错误提示
5. 无论成功失败都会关闭进度态（`finally`）

## 优势

- 用户可以感知“不是卡死，而是在扫描”
- 降低异步上传等待时的焦虑感
- 与后端异步隔离扫描状态保持一致
