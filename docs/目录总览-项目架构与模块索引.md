## 数据库模型关系总览

### accounts（用户）

```
User (abstract_user 基础)
├── 继承自: AbstractUser (Django 内置)
│   ├── username, email, first_name, last_name
│   └── is_staff, is_superuser, is_active
├── Profile (OneToOne → User, related_name: user)
│   ├── avatar (WebP 格式)
│   ├── bio
│   ├── birthday, gender, location
│   ├── last_login_ip
│   └── reputation (信誉分)
└── 反向关联:
    ├── user.posts (Post)
    ├── user.comments (Comment)
    ├── user.topics (Topic)
    ├── user.replies (Reply)
    ├── user.comment_likes (CommentLike)
    ├── user.reply_likes (ReplyLike)
    ├── user.reviewed_comments (ModerationAdmin)
    ├── user.reviewed_topics (ModerationAdmin)
    ├── user.reviewed_replies (ModerationAdmin)
    └── user.reputation (UserReputation)
```

### blog（博客+评论）

```
Category (分类)
└── ← Post (N:1, related_name: categorized_posts)

Tag (标签)
└── ← Post (M:N, through Django built-in / taggit)

Post (文章)
├── ← Comment (1:N, related_name: post)
│   └── ← CommentLike (N:1)
└── ← CommentLike (N:1 → User)

CommentLike (评论点赞)
├── → Comment (N:1, unique_together: (user, comment))
└── → User (N:1)
```

### forum（社区论坛）

```
Board (版块)
└── ← Topic (1:N, related_name: topics)
    └── ← Reply (1:N, related_name: replies)
        └── ← ReplyLike (N:1, unique_together: (user, reply))

ReplyLike (回复点赞)
├── → Reply (N:1)
└── → User (N:1)
```

### moderation（内容审核）

```
SensitiveWord (敏感词)
├── word (敏感词文本)
├── level (严重程度)
└── is_active

ModerationLog (审核日志)
├── content_object (GenericForeignKey: Comment/Topic/Reply)
├── result (通过/拒绝/待审核)
├── reason
├── moderator → User (审核人)
└── moderation_method (AI/百度/人工)

ModerationAdmin (审核管理)
├── moderator → User (审核员)
├── reviewed_comments ← Comment (N:1)
├── reviewed_topics ← Topic (N:1)
└── reviewed_replies ← Reply (N:1)

ModerationReminder (审核提醒)
└── 审核通知管理

UserReputation (用户信誉)
├── user → User (OneToOne)
└── score (信誉分)

ReputationLog (信誉变动日志)
├── user → UserReputation
├── change (变动值)
├── reason (变动原因)
└── created_at
```

### tools（工具箱）

```
ToolConfig (工具配置)
├── tool_slug
├── is_enabled
├── usage_count
└── settings (JSON)

LazyLoadedToolRegistry
└── 动态加载 BaseTool 子类（72 个工具）
```

### core（核心）

```
SiteConfig (站点配置 - 单例模式)
├── site_name
├── site_description
├── site_keywords
├── site_author
├── site_icp (ICP 备案号)
└── site_gongan_beian (公安联网备案号)
```
