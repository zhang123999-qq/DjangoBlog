# v2.3.1 更新说明

**发布日期**: 2026-03-24

## 🚀 性能与安全优化

本次更新专注于**性能优化**和**安全加固**，显著提升系统稳定性和安全性。

### ✨ 新增功能

#### 安全加固
- **CSP 安全头** - 防止 XSS 攻击
- **HSTS 配置** - 强制 HTTPS（生产环境）
- **Cookie 安全** - Secure + HttpOnly + SameSite
- **速率限制** - 防止滥用和攻击

#### 健康检查（Kubernetes）
- `/healthz/` - 综合健康检查
- `/readiness/` - 就绪检查
- `/liveness/` - 存活检查

#### APM 监控
- **Sentry 集成** - 实时异常监控
- Django/Celery/Redis 集成
- 性能追踪和错误报告

### 📈 性能优化

| 优化项 | 收益 |
|--------|------|
| 浏览量 Redis 化 | DB 写入 **-80%** |
| N+1 查询修复 | 查询 **-70%** |
| 工具懒加载 | 性能 **+90%** |

### 🔧 配置更新

新增环境变量：

```bash
# Sentry 监控
SENTRY_DSN=https://xxx@sentry.io/xxx
SENTRY_TRACES_SAMPLE_RATE=0.1
SENTRY_PROFILES_SAMPLE_RATE=0.1

# 安全配置
CSP_ENABLED=true
```

### 📝 升级指南

1. 拉取最新代码
```bash
git pull origin main
```

2. 安装依赖
```bash
uv sync
```

3. 运行迁移（如有）
```bash
uv run python manage.py migrate
```

4. 重启服务
```bash
# 重启 Gunicorn
systemctl restart gunicorn

# 重启 Celery
systemctl restart celery-worker
systemctl restart celery-beat
```
