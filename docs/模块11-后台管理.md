# 模块 11 — 后台管理

> 路径: `apps/core/admin/` | 入口: `config/urls.py` → `admin_site` | 框架: Django Admin（自定义 AdminSite）

---

## 功能概览

定制化的 Django Admin 后台管理系统，**统一注册到自定义 `admin_site`**（非默认 Django admin）。包含自定义仪表盘（业务统计 + 系统信息 + 7 天趋势）、用户管理、文章管理、论坛管理、评论管理、敏感词管理、用户信誉管理、站点配置等完整后台管理功能。

> 设计决策：从原单文件 `admin.py`（约 550 行）拆分为 **11 个模块文件**，按业务域组织，提升可维护性。

@admin.register(User, site=admin_site)
class CustomUserAdmin(UserAdmin):

def has_add_permission(self, request):

## 权限模型

| 角色 | 权限 | 说明 |
|------|------|------|
| **超级管理员 (superuser)** | 所有功能 | 完全访问所有 ModelAdmin |
| **管理员 (staff)** | 根据权限分配 | 需要 `is_staff=True` + 相应 model 权限 |
| **普通用户** | 无法访问 | 不显示 admin 入口 |

> 所有 ModelAdmin 使用 `@admin.register(..., site=admin_site)`，注册到自定义 admin_site，只有 staff/superuser 可访问。
