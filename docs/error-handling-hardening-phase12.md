# 异常处理分层收敛（阶段 12：拒绝原因本地记忆）

覆盖文件：
- `moderation/templates/moderation/dashboard.html`

## 目标

提升审核操作效率：将“最近使用的拒绝原因”本地记忆并可一键复用。

## 本次实现

1. 本地存储键：`moderation.rejectReasons.v1`
2. 拒绝弹层新增“最近使用”区域（最多 5 条）
3. 点击最近原因可一键填入输入框
4. 每次确认拒绝时自动更新最近原因列表（去重 + 最新置顶）

## 说明

- 数据仅存在浏览器 `localStorage`，不上传服务端
- 仅用于提升审核效率，不影响后端逻辑
