什么是 IP 地址？

IP 地址（Internet Protocol Address）是互联网协议为每个网络设备分配的唯一标识符。

IP 地址的分类
类型 格式 说明
IPv4 192.168.1.1 4 段数字，约 43 亿个地址，目前最常用
IPv6 2001:0db8:85a3::1 8 组十六进制数，地址空间近乎无限
公网 IP vs 内网 IP
  • 公网 IP：全球唯一，互联网通信必备
  • 内网 IP：局域网内使用，常见的有 192.168.x.x、10.x.x.x 等
  • NAT：多台设备通过一个公网 IP 上网的技术

常见问题

不同数据源的 IP 数据库更新频率和精度不同，结果可能存在细微差异，一般以 ip-api.com 中文数据为准。

IP 地理位置基于运营商注册信息，不一定精确到当前位置。手机上网通常显示运营商所在地。

请检查输入的 IP 地址格式是否正确。如果是域名，请确保域名能正常解析。API 故障请稍后重试。
